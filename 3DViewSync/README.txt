#  PubSub-3Dの使い方
#
###

### 使い方##############################

  0. node.jsとredisのインストールが必要．
  1. ps3dv0*の中に移動する．
  2. $ redis-server
	ターミナルから実行．
  3. $ node app.js
	ターミナルから実行．
  4. localhost:3000 に接続する．

###
#
# モデルの変換について
#
###

###やり方###############################

 現在はJSON形式のものでしかモデルの
 置き換えができません.

 現状
 0. blender形式のものはblenderで変換
 1. obj形式のものはプログラムから変換
 2. プログラムは3d_model/converter
 3. objファイルをconverterの直下に置く
 4. sh script.sh hoge.obj hoge.js
 5. 上記コマンドで hoge.js に変換できる

###やり方###############################

　テストスクリプトを動かすにはseleniumと
　ChromeDriverのインストールが必要です.

　引数にクライアント数と秒数を渡すことで
　クライアント数分のブラウザを表示して
　システムを動かせます.
